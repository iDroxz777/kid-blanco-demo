# Kid Blanco — demonstração

Protótipo conceitual de uma landing page com recomendação de produtos para a Padaria Kid Blanco.

## Objetivo

Reduzir a indecisão no cardápio com três perguntas rápidas, indicar produtos reais e preparar o pedido para envio pelo WhatsApp.

## Estrutura

- `app/components`: experiência interativa;
- `app/data`: cardápio usado na demonstração;
- `app/lib`: lógica de pontuação e recomendação;
- `app/globals.css`: identidade visual e responsividade.

## Observação

Este projeto é uma demonstração não oficial, construída com informações públicas. Preços, horários, imagens e disponibilidade devem ser confirmados com o estabelecimento antes de uma publicação oficial.
